
# Social Links

An Small Project Based on large scale layout 


## Tech Stack

**Client:** React, Redux, Material ui 

**Server:** Json-Server


## Run Locally

**First Start Json-Server on Port 3030**



Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
